Writing code is no longer the biggest challenge in software development. With modern tools and AI assistance, developers can generate code quickly.

However, what truly defines a developer is:

    How the application is structured

    How decisions are made during development

    How the code is organized for scalability and maintainability

    How different parts of the system communicate with each other

For this assignment, I tried to keep the implementation clean, understandable, and easy to extend, rather than making it unnecessarily complex.
